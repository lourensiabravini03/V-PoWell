# V-PoWell
Simulasi ini memodelkan partikel mikroskopis (seperti elektron) yang terperangkap di dalam Sumur Potensial Tak Hingga 1 Dimensi (Particle in a 1D Box). Model ini adalah fondasi utama untuk memahami fenomena kuantisasi energi, keberadaan energi titik nol (zero-point energy), serta sifat gelombang-partikel dari materi.
